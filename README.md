# Lab Question 2: SQL Database Implementation

This project implements a SQLite database for a university scenario using Python.

## Objective

Practice creating tables, inserting data, and retrieving information with SQL.

## Tech Stack

- Python 3
- SQLite (`sqlite3` module from Python standard library)

## Project Files

- `main.py` - creates the database, inserts records, and runs required queries
- `miva_university.db` - SQLite database file generated by running the script

## Relational Schema

- `Student(student_id, name, email, date_of_birth)`
  - Primary Key: `student_id`
  - Constraint: `email` is `UNIQUE`

- `Instructor(instructor_id, name, department, phone_number)`
  - Primary Key: `instructor_id`

- `Course(course_code, title, credit_hours, department, instructor_id)`
  - Primary Key: `course_code`
  - Foreign Key: `instructor_id` references `Instructor(instructor_id)`
  - Constraint: `instructor_id` is `NOT NULL`

- `Enrollment(student_id, course_code)`
  - Composite Primary Key: `(student_id, course_code)`
  - Foreign Keys:
    - `student_id` references `Student(student_id)`
    - `course_code` references `Course(course_code)`

## Data Population

The script inserts at least 20 records into each table:

- `Student`: real student records (provided) + generated records to reach 20
- `Instructor`: 20 instructors with real-sounding names
- `Course`: real course titles/codes plus a `Database Systems` course (for Query B)
- `Enrollment`: 20 enrollments linking students to courses

## Required SQL Queries Implemented

1. List all students and their enrolled courses.
2. Find the instructor teaching `"Database Systems"`.
3. Count how many students are enrolled in each course.

## How to Run

From the project folder:

```bash
python3 main.py
```

The script will:

1. Drop and recreate all tables
2. Insert fresh records
3. Print outputs for Query A, Query B, and Query C

## Notes

- SQLite foreign keys are enabled with `PRAGMA foreign_keys = ON`.
- Running the script resets the data each time because the tables are dropped first.
